## 🚀 快速上手

### 界面布局

启动应用后，你会看到：

- **左侧边栏**：品牌标识 "K" + 五大板块导航（剧本、角色、场景、导演、S级）
- **主工作区**：当前板块的操作界面
- **底部工具栏**：帮助、设置、主题切换

### 第一次使用

1. **配置 API**：点击左下角「设置」→「API 配置」，添加你的 AI 服务商密钥
2. **创建项目**：在 Dashboard 点击「新建项目」
3. **开始创作**：按照 剧本 → 角色 → 场景 → 导演 → S级 的流程进行

## 🎬 工作流程

### 标准创作流程

```
📝 剧本板块
   ↓ 粘贴剧本 → 智能解析 → 生成场景和分镜

🎭 角色板块
   ↓ 创建角色 → 设置角色圣经 → 上传参考图

🌄 场景板块
   ↓ 生成场景图 → 多视角联合图

🎬 导演板块
   ↓ 调整分镜 → 设置摄影参数 → 批量生图/生视频

⭐ S级板块
   ↓ Seedance 2.0 多模态创作 → 多镜头合并 → 导出成片
```

## 📚 五大板块详解

### 1️⃣ 剧本板块 (Script)

**功能**：智能解析剧本，自动拆分场景和分镜

#### 使用步骤

1. **粘贴剧本**
   - 支持标准剧本格式（场景、人物、对白、动作）
   - 支持多集/多幕结构

2. **智能解析**
   - 点击「解析剧本」按钮
   - 系统自动识别：
     - 场景（时间、地点）
     - 角色（出场人物）
     - 对白（台词）
     - 动作描述
     - 镜头语言

3. **场景拆分**
   - 自动按场景分组
   - 每个场景包含多个分镜
   - 可手动调整分镜切割点

4. **分镜切割**
   - 根据动作和对白自动切割分镜
   - 每个分镜包含：
     - 画面描述
     - 角色动作
     - 台词对白
     - 镜头参数建议

#### 剧本格式示例

```
场景1：咖啡厅 - 白天 - 内景

张三坐在窗边，手里拿着一杯咖啡。

张三：（看向窗外）今天天气真好。

李四推门进来，走向张三。

李四：（微笑）好久不见！

张三站起来，两人握手。
```

#### 使用技巧

- **清晰的场景标记**：使用「场景X：地点 - 时间 - 内/外景」格式
- **明确的角色名**：每次对白前标注角色名
- **详细的动作描述**：有助于生成更准确的分镜
- **镜头语言提示**：可在描述中加入「特写」「全景」等关键词

### 2️⃣ 角色板块 (Characters)

**功能**：管理角色一致性，确保同一角色在不同分镜中外观统一

#### 核心概念：角色圣经 (Character Bible)

角色圣经是角色的"身份证"，包含 6 层身份锚点：

1. **基础信息**：姓名、年龄、性别
2. **外貌特征**：身高、体型、发型、肤色
3. **服装风格**：常穿衣服、配饰
4. **性格特点**：性格、习惯、口头禅
5. **背景故事**：职业、经历
6. **视觉风格**：画风、色调

#### 使用步骤

1. **创建角色**
   - 点击「添加角色」
   - 输入角色名称

2. **编写角色圣经**
   - 填写 6 层身份锚点
   - 越详细越好，有助于保持一致性

3. **上传参考图**
   - 点击「上传参考图」
   - 支持多张参考图
   - 系统会在生成时自动引用

4. **角色绑定**
   - 在分镜中选择角色
   - 系统自动应用角色圣经

#### 角色圣经示例

```
角色名：张三
年龄：28岁
性别：男

外貌特征：
- 身高 175cm，中等身材
- 短发，黑色，略微凌乱
- 戴黑框眼镜
- 肤色偏白

服装风格：
- 常穿白色衬衫 + 黑色休闲裤
- 戴银色手表
- 背黑色双肩包

性格特点：
- 内向、细心
- 喜欢思考
- 说话语速较慢

背景故事：
- 程序员，在互联网公司工作
- 喜欢咖啡和阅读

视觉风格：
- 2D 动漫风格
- 色调偏冷
```

#### 使用技巧

- **参考图质量**：使用高清、正面、光线充足的图片
- **描述具体化**：避免模糊描述，如「帅气」改为「高鼻梁、深邃眼睛」
- **风格统一**：所有角色使用相同的视觉风格（2D/3D/写实等）

### 3️⃣ 场景板块 (Scenes)

**功能**：生成场景背景图，支持多视角联合图

#### 使用步骤

1. **场景列表**
   - 从剧本板块自动导入场景
   - 显示场景名称、时间、地点

2. **生成场景图**
   - 选择场景
   - 点击「生成场景图」
   - 系统根据场景描述自动生成

3. **多视角联合图**
   - 生成同一场景的不同视角
   - 如：咖啡厅的全景、窗边特写、吧台视角
   - 有助于后续分镜使用

4. **场景编辑**
   - 可手动调整场景描述
   - 重新生成场景图

#### 场景描述示例

```
场景：咖啡厅 - 白天 - 内景

描述：
- 温馨的咖啡厅，木质装修
- 大落地窗，阳光洒进来
- 窗边有几张小圆桌
- 吧台在左侧，摆放着咖啡机
- 墙上挂着艺术画
- 整体色调温暖
```

#### 使用技巧

- **详细描述**：包含光线、色调、氛围
- **视角提示**：标注「俯视」「平视」「仰视」
- **风格统一**：与角色板块保持相同视觉风格

### 4️⃣ 导演板块 (Director)

**功能**：专业分镜系统，批量生图/生视频

#### 核心功能

1. **分镜管理**
   - 查看所有分镜
   - 调整分镜顺序
   - 编辑分镜内容

2. **摄影参数**
   - **景别**：特写、近景、中景、全景、远景
   - **机位**：平视、俯视、仰视、侧面
   - **运动方式**：固定、推拉、摇移、跟随

3. **视觉风格**
   - 2D 动漫
   - 3D 渲染
   - 写实风格
   - 定格动画
   - 水彩画风
   - 等等...

4. **批量生成**
   - 批量生图：一键生成所有分镜的图片
   - 批量生视频：一键生成所有分镜的视频

#### 使用步骤

1. **选择分镜**
   - 在分镜列表中选择要处理的分镜

2. **设置摄影参数**
   - 选择景别（如：中景）
   - 选择机位（如：平视）
   - 选择运动方式（如：固定）

3. **选择视觉风格**
   - 点击「视觉风格」下拉菜单
   - 选择合适的风格

4. **生成图片**
   - 单个生成：点击分镜卡片的「生成图片」
   - 批量生成：点击顶部「批量生图」

5. **生成视频**
   - 单个生成：点击分镜卡片的「生成视频」
   - 批量生成：点击顶部「批量生视频」

#### 摄影参数说明

**景别**：
- **特写 (Close-up)**：脸部或物体细节
- **近景 (Medium Close-up)**：胸部以上
- **中景 (Medium Shot)**：腰部以上
- **全景 (Full Shot)**：全身
- **远景 (Long Shot)**：人物 + 环境

**机位**：
- **平视 (Eye Level)**：与人物视线平齐
- **俯视 (High Angle)**：从上往下拍
- **仰视 (Low Angle)**：从下往上拍
- **侧面 (Side Angle)**：从侧面拍

**运动方式**：
- **固定 (Static)**：镜头不动
- **推拉 (Dolly)**：镜头前后移动
- **摇移 (Pan/Tilt)**：镜头左右/上下转动
- **跟随 (Follow)**：镜头跟随人物移动

#### 使用技巧

- **景别搭配**：远景建立环境 → 中景展示动作 → 特写表现情绪
- **机位变化**：避免连续使用相同机位，增加视觉变化
- **运动节奏**：静态镜头为主，运动镜头点缀
- **风格统一**：整个项目使用相同视觉风格

### 5️⃣ S级板块 (Sclass - Seedance 2.0)

**功能**：Seedance 2.0 多模态创作，多镜头合并生成连贯视频

#### 核心功能

1. **多模态引用**
   - @Image：引用图片（角色参考图、场景图、首帧图）
   - @Video：引用视频片段
   - @Audio：引用音频（背景音乐、音效）

2. **多镜头合并**
   - 将多个分镜分组
   - 合并生成连贯叙事视频
   - 自动处理镜头转场

3. **智能提示词构建**
   - 自动三层融合：
     - 动作描述
     - 镜头语言
     - 对白唇形同步

4. **首帧图网格拼接**
   - 多张首帧图按 N×N 网格拼接
   - 确保视频连贯性

#### 使用步骤

1. **选择分镜**
   - 在 S级板块查看所有分镜
   - 选择要合并的分镜（可多选）

2. **分组合并**
   - 点击「创建分组」
   - 将选中的分镜添加到分组
   - 一个分组生成一个连贯视频

3. **添加多模态引用**
   - **@Image**：
     - 自动收集角色参考图
     - 自动收集场景图
     - 自动收集首帧图
   - **@Video**：
     - 引用已生成的视频片段
   - **@Audio**：
     - 上传背景音乐
     - 上传音效

4. **编辑提示词**
   - 系统自动生成提示词
   - 可手动调整优化
   - 确保提示词 ≤ 5000 字符

5. **生成视频**
   - 点击「生成视频」
   - 等待 Seedance 2.0 处理
   - 下载生成的视频

#### Seedance 2.0 参数限制

- **图片**：最多 9 张
- **视频**：最多 3 个
- **音频**：最多 3 个
- **提示词**：≤ 5000 字符

系统会自动校验，超出限制会提示调整。

#### 使用技巧

- **分组策略**：按场景或情节分组，每组 3-5 个分镜
- **首帧图质量**：使用高质量、构图清晰的首帧图
- **提示词优化**：突出关键动作和情绪，删除冗余描述
- **音频匹配**：背景音乐与视频节奏匹配

## ⚙️ API 配置

### 添加 API 服务商

1. **进入设置**
   - 点击左下角「设置」图标
   - 选择「API 配置」标签

2. **添加服务商**
   - 点击「添加服务商」
   - 选择服务商类型（如：OpenAI、Anthropic、国内服务商等）
   - 输入 API Key
   - 点击「保存」

3. **功能绑定**
   - 为不同功能绑定不同的服务商
   - 如：
     - 剧本解析 → GPT-4
     - 图像生成 → Midjourney
     - 视频生成 → Seedance 2.0

4. **API Key 轮询**
   - 可添加多个相同服务商的 API Key
   - 系统自动负载均衡
   - 失败自动重试

### 推荐配置

- **剧本解析**：GPT-4 或 Claude
- **角色圣经生成**：GPT-4 或 Claude
- **图像生成**：Midjourney、Stable Diffusion、DALL-E
- **视频生成**：Seedance 2.0、Runway、Pika

## ❓ 常见问题

### Q1：剧本解析失败怎么办？

**A**：
- 检查剧本格式是否规范
- 确保场景标记清晰（场景X：地点 - 时间）
- 检查 API Key 是否有效
- 尝试分段解析（每次解析 1-2 个场景）

### Q2：角色一致性不好怎么办？

**A**：
- 完善角色圣经，增加详细描述
- 上传高质量参考图（正面、光线充足）
- 在提示词中强调角色特征
- 使用相同的视觉风格

### Q3：生成的图片/视频不满意怎么办？

**A**：
- 调整摄影参数（景别、机位、运动方式）
- 优化提示词，增加细节描述
- 更换视觉风格
- 重新生成（可多次尝试）

### Q4：批量生成失败怎么办？

**A**：
- 检查 API Key 额度是否充足
- 检查网络连接
- 减少批量数量，分批生成
- 查看任务队列，手动重试失败任务

### Q5：Seedance 2.0 提示参数超限怎么办？

**A**：
- 减少引用的图片/视频/音频数量
- 精简提示词，删除冗余描述
- 将分镜拆分成多个分组

### Q6：如何导出最终成片？

**A**：
- 在 S级板块生成所有分组视频
- 使用视频编辑软件（如 Premiere、Final Cut）合并
- 添加转场、字幕、音效
- 导出最终成片

## 💡 使用技巧

### 提高效率

1. **模板复用**
   - 保存常用的角色圣经模板
   - 保存常用的场景描述模板
   - 保存常用的摄影参数组合

2. **批量操作**
   - 使用批量生图/生视频功能
   - 一次性处理多个分镜
   - 利用 API Key 轮询提高并发

3. **快捷键**
   - `Ctrl/Cmd + S`：保存
   - `Ctrl/Cmd + Z`：撤销
   - `Ctrl/Cmd + Shift + Z`：重做

### 提高质量

1. **剧本阶段**
   - 详细的场景描述
   - 清晰的角色动作
   - 明确的镜头语言提示

2. **角色阶段**
   - 完善的角色圣经
   - 高质量参考图
   - 统一的视觉风格

3. **导演阶段**
   - 合理的景别搭配
   - 丰富的机位变化
   - 适当的运动镜头

4. **S级阶段**
   - 精准的多模态引用
   - 优化的提示词
   - 合理的分组策略

### 节省成本

1. **API 选择**
   - 剧本解析使用便宜的模型（如 GPT-3.5）
   - 图像生成使用性价比高的服务商
   - 视频生成谨慎使用（成本较高）

2. **生成策略**
   - 先生成关键分镜，确认效果后再批量生成
   - 使用低分辨率预览，满意后再生成高清
   - 合理使用重试次数

3. **资源管理**
   - 定期清理无用的生成结果
   - 备份重要的角色圣经和场景图
   - 导出项目数据，避免丢失
